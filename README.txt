Get twitter feeds by twitter username and display it with shortcodes.
== Description ==

Ulitmate twitter feed is the best plugin for displaying twitter feeds by username, you can display them with already designed template by shortcode, or you can display them in your ready desgined template with the WP REST API.

Some of the awesome features include:
*   Create unlimted feeds with shortcodes
*   Create unlimted feeds with WP REST API URLS
*   Already Desgined Template for displaying your feeds
*   You can display your feeds in your already designed template
*	You can display your feeds with widgets

== Installation ==

1. Upload `ultimate-twitter-feed.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the generated shortcodes in your templates
